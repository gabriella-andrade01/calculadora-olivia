# Calculadora Olivia 🐕

Calculadora web com tema **Galgo Italiano** — elegante, afetiva e completamente funcional.

## Funcionalidades

- Operações aritméticas: +, −, ×, ÷
- Porcentagem (%), inversão de sinal (±), backspace (⌫), limpar (C)
- Operações encadeadas com resultado intermediário
- Destaque visual do operador ativo
- Suporte a teclado completo
- Interface responsiva (mobile e desktop)

## Design

- Paleta bege e marrom inspirada na pelagem do galgo italiano
- Silhueta SVG do galgo inline (zero dependências de imagem)
- Tipografia *Cormorant Garamond* com fallback Georgia
- WCAG AA — acessibilidade garantida

## Como usar

Abra diretamente no browser:

```bash
# Opção 1: arquivo local
open index.html

# Opção 2: servidor local simples
npx serve .
```
